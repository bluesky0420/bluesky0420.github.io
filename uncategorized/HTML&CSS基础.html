HTML&CSS

概述

HTML & CSS概述

1.web

​ 互联网

2.W3C

​ 万维网联盟

​ 为互联网提供各种标准

3.xml

​ 可扩展的标记语言：extension markup language,用于定义文档结构。

4.什么是HTML

​ HTML是W3C定义的语言标准：用来描述页面结构的语言。

​ 结构：有什么东西，这个东西表示什么含义。

​ HTML：Hyper Text Markup Language，超文本标记语言

5.什么是CSS

​ CSS是W3C定义的语言标准：用来展示页面效果的语言。

​ CSS决定了页面长什么样子。

6.执行HTML CSS

​ HTML、CSS → 浏览器内核执行 → 页面

​ 浏览器：shell(外壳) + core(内核):js执行引擎、渲染引擎、其他模块

​ 主流浏览器及其内核：

​ IE：Trident

​ Chrome：webkit(前) blink(现)

​ Firefox：Gecko

​ Safari：Webkit

​ Opera：Presto(前) blink(现)

7.版本和兼容

​ HTML5：2014年制定完成

​ CSS3：至今还没有制定完成

​ XHTML：完全符合XML规范的HTML的一个版本

1.显示文件扩展名

2.安装浏览器

​ IE，chrome，Firefox，safari，opera (推荐chrome)

​ 记事本(推荐前期使用记事本)，sublime，vscode(插件多)，Atom，Dw(不推荐)，HBuilder

语义化

1.什么是语义化

​ 每一个HTML元素都具体有具体含义

​ 所有HTML元素与展示在页面的效果无关,页面效果由CSS决定

​ 当我们只有HTML元素而没有设置CSS属性但页面仍有效果是因为浏览器有默认的CSS样式

2.为什么需要语义化

​ 为了搜索引擎优化(SEO)

​ 每隔一段时间搜索引擎会从整个网页获取网页源码

​ 为了让浏览器理解网页

​ 阅读模式、语音模式

1.h元素

​ 标题元素h1-h6

2.p元素

​ 文本元素，paragraphs

3.span元素

​ 没有语义，用来设置样式

4.pre元素

​ 预格式化文本元素

​ 空白折叠：在源码中连续的空白字符，在页面显示会被折叠成一个空格

​ 在pre元素中不会出现空白折叠，通常用于显示页面中的一些代码

​ pre元素的本质：他有一个默认的css属性white-space：pre;

路径的写法

1.站内资源和站外资源

​ 站内资源：当前网站的资源

​ 站外资源：非当前网站的资源

2.绝对路径和相对路径

​ 绝对路径书写格式：

​ url地址:

​ 协议名://主机号:端口/路径

​ 协议名：https、http、file….

​ 主机号：域名、IP地址

​ 端口号：http协议默认端口是80，https协议默认端口号是443。

​ 相对路径：

​ 以./开头./表示当前目录所在资源

​ 站内资源：使用相对路径和绝对路径都可以

​ 站外资源：绝对路径

图片元素

1.img元素

​ image缩写，空元素

​ src(source)属性：图像资源地址

​ alt属性：当图片资源失效时，将使用该元素文字代替图片

2.和a元素联用

​ 在a元素中添加img元素

3.和map元素联用

​ map：地图，可将一张图片的各个区域分割成不同的链接地址

​ map子元素：area

​ circle：园形 （圆心坐标，半径）

​ rect：矩形 （左上角坐标，右下角坐标）

​ poly：多边形 （各个坐标点的坐标）

4.figure元素

​ 将图片、图片标题、图片描述包裹起来

​ 子元素：figcaption

多媒体元素

1.video：视频

​ 属性autoplay：自动播放，muted：静音播放，controls：控制控件显示，loop：循环播放

2.audio：音频

​ 属性autoplay：自动播放，muted：静音播放，controls：控制控件显示，loop：循环播放

3.兼容性

​ 旧版本浏览器不支持这两个属性

​ 不同版本的浏览器支持的音频格式可能不同

列表元素

1.有序列表

​ ol：ordered list

​ li：list item

​ ol的属性，type值为1时，按数字大小排列；为i时，按罗马数字排序，为A，a时，按大写小写排序

​ 不建议使用type属性，在css属性的list-style-type可以进行修改

2.无序列表

​ ul：unordered list

​ li：list itme

​ 通常用于制作菜单

3.定义列表

​ 用于一些专业术语的定义

​ dl：definition list

​ dt：definition title

​ dd：definition description

容器元素

1.div元素

​ 没有语义，用来定义块级元素

2.语义化元素

​ header：通常表示页头或者文章的头部

​ article：通常表示整篇文章

​ section：通常表示文章的章节

​ footer：通常用于表示页尾，或表示文章的尾部

​ aside：通常用于表示附加信息，侧边栏

元素的包含关系

1.之前的说法

​ 之前的说法是，会计元素可以包含行级元素，行级元素不能包含块级元素，a元素除外

​ 现在因为语义化的提出，上面的说法失效，因为html元素与它展现页面的效果无关

2.总结

​ 1.容器元素中可以包含任何元素

​ 2.a元素中可以包含几乎所有元素

​ 3.某些元素有固定的子元素如：ul > li…..

​ 4.标题元素和段落元素不能互相嵌套，且不能包含容器元素
