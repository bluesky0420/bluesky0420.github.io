VSCode

【注】：博主使用的VSCode版本为1.86.1

命令行

  • 先决条件：安装VSCode软件时将其加入环境变量PATH中
  • 命令：code 选项 目录/文件路径
    • 常用命令
      • code -h：获取code命令帮助
      • code -r path：窗口复用（关闭当前窗口，新建窗口打开指定文件）
      • code -r -g path:number：窗口复用并定位到指定文件的指定行（可用于当代码报错时定位到指定位置）
      • code -r -d file1path file2path：窗口复用并对比两个文件内容
    • 与管道联用：ls | code 选项 目录/文件路径
    • 扩展管理（Extensions management）
    • 故障排除（Troubleshooting）

文本操作

  • 光标移动
    • 单个字符移动（移动单个字符）：左/右方向键
    • 单个单词移动（移动单个单词）：Ctrl+左/右方向键
    • 行内移动（移动到行首/尾）：HOME/END
    • 多行移动：上/下方向键
    • 代码块移动（通过匹配括号进行移动）：Ctrl+Shift+\
    • 整个文档移动（移动到文档首/尾）：Ctrl+HOME/END
    • 撤销光标移动（移动光标后希望光标退回到原来位置）：Ctrl+U
  • 文本选择
    • 单个字符选择：Shift+上/下/左/右方向键 或者 鼠标左键单击当前光标所在字符
    • 单个单词选择：Ctrl+Shift+左/右方向键 或者 鼠标左键双击当前光标所在单词
    • 单行选择：Ctrl+HOME/END 或者 鼠标左键三击当前光标所在行 或者 鼠标左键单击文本左侧行号
    • 多行选择：Shift+上/下方向键 或者 Ctrl+Shift+HOME/END 或者 鼠标左键单击文本左侧行号并向上或向下拖动选中指定文本
    • 全选：Ctrl+A 或者 鼠标左键四击
  • 文本移动
    • 普通方式：选中文本 剪切文本 粘贴文本
    • 单行移动：Alt+上/下方向键
    • 多行移动：选中指定的文本后 使用ALT+上/下方向键
    • 鼠标操作：选中指定文本后 拖动到指定位置
  • 文本复制
    • 普通方式：选中文本 复制文本 粘贴文本
    • 单行复制：Alt+Shift+上/下方向键
    • 多行复制：选中指定文本后 使用Alt+Shift+上/下方向键
    • 鼠标操作：选中指定文本后 按住Ctrl键后拖动文本到指定位置
  • 文本删除
    • 普通方式：选中文本后按下回车或者delete键
    • 文本行删除：Ctrl+Shift+K（删除文本） 或者 Ctrl+X（剪切文本）
  • 换行符
    • 普通换行：Enter
    • 当行下添加换行：Ctrl+Enter
    • 当行上添加换行：Ctrl+Shift+Enter
  • 缩进
    • 单行缩进：Ctrl+[/]
    • 多行缩进：选中指定文本后 使用 Ctrl+[/]
  • 制表符
    • 正向制表符：tab
    • 反向制表符：Shift+tab

光标操作

  • 多光标
    • 使用鼠标：按住鼠标中键向上/下拖动
    • 使用键盘
      • 当前光标位置垂直向上或向下创建多光标：Ctrl+Alt+上/下方向键
      • 匹配相同内容并在其后创建光标：Ctrl+D
    • 鼠标+键盘：Alt+鼠标左键单击指定位置

快捷键

【说明】：以下快捷键均为默认快捷键，更多自定义快捷键可通过Ctrl+Shift+P打开命令面板，搜索keyboard shortcuts进行自定义设置。

  • 打卡设置界面：Ctrl+,
  • 活动栏
    • 文件资源管理：Ctrl+Shift+E
    • 跨文件搜索：Ctrl+Shift+F
    • 源代码管理：Ctrl+Shift+G
    • 启用和调试：Ctrl+Shift+D
    • 扩展管理：Ctrl+Shift+X
  • 命令面板：F1/Ctrl+Shift+P
  • 底部状态栏
    • 底部面板（默认位置，可调整）-问题：Ctrl+Shift+M
    • 底部面板（默认位置，可调整）-输出：Ctrl+Shift+U
    • 底部面板（默认位置，可调整）-调试控制台：Ctrl+Shift+Y
    • 底部面板（默认位置，可调整）-终端：Ctrl+`
  • 编辑器
    • 放大编辑器界面（包括字体图标等）：Ctrl++
    • 缩小编辑器界面（包括字体图标等）：Ctrl+-
  • 代码
    • 代码格式化：Alt+Shift+F
    • 代码注释：Ctrl+/
    • 代码自动补全
      • 补全代码：输入关键字后弹出自动补全窗口可选择想要的函数后按下tab自动补全代码
      • 隐藏代码补全窗口：Escape
      • 隐藏代码不全窗口后显示窗口：Ctrl+空格键
      • 重构
        • 变量或者函数名全局修改：选中变量名F2，在弹出的窗口中修改
        • 将函数中的某一块代码提取成单独的函数：选中代码块，在黄色的小灯泡选择抽离成全局函数或父函数中的子函数
      • 折叠代码块
        • 折叠：Ctrl+Shift+{
        • 展开：Ctrl+Shift+}
  • 跳转
    • 文件跳转（同一个项目下多个文件跳转）：Ctrl+P
    • 段落跳转（文件的指定行跳转）：Ctrl+G
    • 符号跳转（代码的指定函数跳转）：Ctrl+Shift+O
    • 函数调用&实现跳转：F12/Ctrl+F12/Ctrl+鼠标左键
    • 函数引用跳转：Shift+F12
  • 分屏
    • 横向拆分：Ctrl+\
    • 纵向拆分：Ctrl+k Ctrl+\

命令面板

【说明】：命令面板提供了一套帮助我们进行快速操作的命令，且这些命令绑定快捷键，通过命令和快捷键能提高我们的工作效率

  • 行排序
    • 按升序排列行：sort lines Ascending
    • 按降序排列行：sort lines Descending
  • 合并行：join lines
  • 转换为大/小写
    • 转换为大写：transform to uppercase
    • 转换为小写：transform to lowercase
  • 转置光标处字符：transpose characters around the cursor
  • 缩进：indent

配置文件

  • 代码补全窗口：”editor.quickSuggestions”
  • 参数预览窗口：”editor.parameterHints”
  • 面包屑导航：”breadcrumbs.enabled”
  • 小地图：”editor.minimap”

VSCode配置文件备份方案

  • 使用VSCode自带的云端备份功能，登陆微软账号或者GitHub账号，登陆后即可设置同步
  • 导出配置文件到本地，文件-共享-导出配置文件
  • 导出配置文件到GitHub gist
